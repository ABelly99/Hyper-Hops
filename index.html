About Hyper Hops; Experience Parkour Like Never Before

Hyper Hops is coming soon for the public. It is going to be a Minecraft server
featuring numerous easy, medium, and hard parkour maps. But it's NOT A BORING GENERIC PARKOUR SERVER.
There will be power-ups, speed boosts, jump boosts, special custom blocks, and
many more unique features that set it apart from ordinary parkour maps!
